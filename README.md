# Template Project

This is a template project based on the Kospi200 project configuration.
It includes a minimal setup with React, TypeScript, Webpack, and @imspdr/ui.

## Getting Started

1.  Navigate to the directory:
    ```bash
    cd template
    ```
2.  Install dependencies:
    ```bash
    yarn install
    # or
    npm install
    ```
3.  Start the development server:
    ```bash
    yarn dev
    # or
    npm run dev
    ```

## Project Structure

-   `src/`: Source code
    -   `components/`: Reusable components
    -   `pages/`: Page components
    -   `hooks/`: Custom hooks
-   `public/`: Static assets
-   `webpack.config.js`: Webpack configuration
-   `tsconfig.json`: TypeScript configuration

## Technologies

-   React
-   TypeScript
-   Webpack
-   @imspdr/ui
-   Emotion (Styled Components)
-   React Query
